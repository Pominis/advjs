# 环境配置

- [Node.JS](https://nodejs.org/en/download/) LTS
- [pnpm](https://pnpm.io/installation) 包管理器

```bash
# clone project
git clone https://github.com/YunYouJun/advjs
cd advjs
```

```bash
# install dependencies, must use pnpm
pnpm install
```

## 常用命令

- `pnpm dev` - 启动开发环境（stub）
- `pnpm dev:plugins` - 启动插件开发环境
